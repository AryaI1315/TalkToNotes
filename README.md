# TalkToNotes
TalkToNotes is a simple and efficient Android application that converts your speech into text notes using built-in speech recognition. It allows users to quickly capture ideas, save notes with timestamps, and export them as a PDF.

✨ Features
🎤 Voice to Text
Convert spoken words into text in real-time using speech recognition.
📝 Smart Note Formatting
Automatically formats sentences and replaces words like “comma”, “full stop”, “new line”.
🕒 Date & Time Stamps
Each note is saved with the current date and time for easy tracking.
💾 Save Notes Locally
Notes are stored using a local database for offline access.
📄 Export to PDF
Generate a PDF file containing all saved notes.
🗑️ Delete Notes
Remove notes easily with a long press.

🛠️ Tech Stack
Java (Android)
SpeechRecognizer API
SQLite Database
PDFDocument API
Android UI (XML)

📸 UI Highlights
Clean and simple interface
Voice input button
Organized note list with timestamps
Modern card-style layout (optional improvement)

🚀 How It Works
Tap the 🎤 button to start speaking
Your speech is converted into text
Edit (if needed) and save the note
Notes are stored with date & time
Export all notes as a PDF anytime
