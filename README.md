# 📱 TalkToNotes – Voice to Notes App

**TalkToNotes** is a simple and efficient Android application that converts your speech into text notes using built-in speech recognition. It helps users quickly capture ideas, save notes with timestamps, and export them as a PDF.

---

## ✨ Features

- 🎤 **Voice to Text**
  - Convert speech into text in real-time using Android SpeechRecognizer

- 📝 **Smart Formatting**
  - Automatically replaces words like:
    - "new line" → line break  
    - "comma" → ,  
    - "full stop" → .  

- 🕒 **Date & Time Stamps**
  - Each note is saved with the current date and time

- 💾 **Local Storage**
  - Notes are stored using SQLite database

- 📄 **Export to PDF**
  - Generate a PDF file containing all saved notes

- 🗑️ **Delete Notes**
  - Long press to delete notes easily

---

## 🛠️ Tech Stack

- Java (Android)
- SpeechRecognizer API
- SQLite Database
- PDFDocument API
- XML UI Design

## 🚀 How to Use

1. Tap 🎤 **Start Recording**
2. Speak your note clearly
3. Text appears on screen
4. Tap **Save** to store the note
5. Tap **PDF** to export all notes
